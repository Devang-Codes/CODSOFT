# Movie Recommendation System

## Description

A content-based movie recommendation system built using Machine Learning.

The system recommends movies based on genre similarity using CountVectorizer and Cosine Similarity.

## Features

- Movie Search
- Top 5 Recommendations
- Content-Based Filtering
- GUI Interface using Tkinter
- Machine Learning Recommendation Engine

## Technologies

- Python
- Pandas
- Scikit-Learn
- Tkinter

## Run

CLI Version

```bash
python recommender.py
```

GUI Version

```bash
python recommender_gui.py
```

## Concepts Used

- Machine Learning
- NLP Vectorization
- Cosine Similarity
- Recommendation Systems
